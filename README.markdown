# ♟️ Chess Game

A fully functional browser-based Chess Game built using HTML, CSS, and JavaScript.

## Features
- ♟️ Complete chess rules
- 🔄 Piece movement validation
- 👑 Check and Checkmate detection
- 🎨 Responsive UI
- 💻 Play directly in the browser

## Technologies Used
- HTML5
- CSS3
- JavaScript
